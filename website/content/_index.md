Small exercises to get you used to reading and writing C code. Cling is inspired and forked from [rustlings](https://github.com/rust-lang/rustlings)

<script src="https://asciinema.org/a/719805.js" id="asciicast-719805" async="true"></script>

## Quick start

```bash
# Installation
cargo install clings
# Initialization
clings init
# Moving into new directory
cd clings
# Starting Clings
clings
```
